#  Projeto AdventureWorks - Análise de Vendas SQL

Este repositório contém um conjunto de análises de Business Intelligence utilizando o banco de dados **AdventureWorksDW2022**.

##  O que você encontrará aqui:
- Consultas SQL analíticas (T-SQL)
- Uso de Window Functions (`RANK`, `LAG`, `SUM OVER`)
- KPIs de vendas, faturamento e fidelização de clientes

##  Estrutura do Repositório
- [**SQL/**](./SQL): Pasta contendo todos os scripts `.sql` e a documentação detalhada de cada consulta.
